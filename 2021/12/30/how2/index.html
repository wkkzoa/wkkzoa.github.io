这个网站怎么来的？（二）

建站
技术
发布日期:   2021-12-30
文章字数:   1.6k

你可以随心所欲地决定自己网站的专属网址——只要你付得起钱。当然网站的主域名还是xxx.github.io，绑定了你自己选择的域名后，两个域名都可以跳转至你的网站。这篇还会介绍一些hexo主题、如何新建一篇文章以及工具的选择，SEO我还没学，也暂时没想法。不过有了这些，就已经能实现主要功能了。

绑定域名

你需要找域名注册商。国内腾讯云、阿里云、华为云都可以，国外Namesilo比较划算。这里我以阿里云（万网）为例：

首先登录万网（可以用支付宝账号登录），在搜索栏处查询你想要的域名和价格，一般而言.com结尾的域名会比较贵，短域名也会比较贵，如下图所示：

这一步就到这里吧，怎么买，跟着点，跟着付钱就行了。反正我的域名只花了1块钱，租了一年。

然后，需要验证邮箱和实名认证，这些一般当天就能完成，需要上传身份证人像面，这个也跟着来就行了。

接下来，进入你的阿里云控制台（网页端在右上角）,找到域名，点击解析。

如何查询某网站的IPV4地址？

win+R打开运行输入cmd回车，打开命令提示符，输入

ping xxx.github.io

其中xxx即你的Github用户名，xxx.github.io即主域名，下图红圈部分即该网站的IPV4地址。

HEXO主题

目前我所使用的是next主题。你可以在Git中敲代码下载，也可以去作者的github上下载压缩包解压到你的themes文件夹内。接下来我说明的是前一种。

打开你的themes文件夹，而不是根目录blog，右键Git Bash here输入如下命令：

git clone https://github.com/theme-next/hexo-theme-next

该命令的意思是从该网站克隆该主题，如果网络抽风，从浏览器输入该网址下载其压缩包也行，有时候两种方式网络都不行，等吧。

用记事本或者vscode打开blog下的_config.yml，按如下格式修改主题：

# Site
title: 你网站的名字                     #标题
subtitle: ''
description: 上次读到这句话还是在上次。    #简介或者格言
keywords:
author: SB                            #作者
language: zh-CN                       #主题语言
timezone: Asia/Shanghai               #中国的时区，这里就用Shanghai，不要输入你的住址，可能是无效的

# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
theme: next                           #主题改为next

打开目录blog/themes/next/下的_config.yml，_

日后你会经常在这两个_config.yml中切换，别搞错了。

找到# Schemes项，删除你想用的主题前的#号即可。四个主题只能同时用一个。退出保存，输入hexo clean，hexo g指令。

接下来若是输入hexo s，浏览器输入默认本地端口localhost:4000查看效果，记得按ctrl+c退出。

输入hexo d，即上传至仓库，一段时间后进入你的个人网站，按shift+F5键可以删缓存刷新，查看页面是否更改。

那么还有哪些主题可以选择呢？这个自行在网上找吧。B站上找会比较方便，可以直接通过视频看到效果，csdn上会比较详细点。或者直接百度hexo主题去查，或者去其它个人网站上学习，等等。

第一篇个人文章

打开blog文件夹，右键Git Bash here，输入命令hexo new “xxx”，意思是新建一个命为xxx.md的（后缀即为makedown的缩写）文档。

打开\blog\source_posts，你能看到自带的helloworld.md，下面就是你刚刚生成的xxx.md，按照自带文档的内容依葫芦画瓢，去改动你的xxx.md，你就知道该怎么写文章、添加链接和标题了。

工欲善其事 必先利其器

一直用记事本，不管是写文章还是改代码，其实不太方便，也可能遇到兼容问题，推荐几个工具。

用于打开_config.yml的工具，功能齐全堪比IDE——Visual Studio Code，微软出品，兼容市面上大部分的文档，如果官网下载比较慢，可以在国内的软件市场下载，但要小心捆绑。
